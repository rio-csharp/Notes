# Learning Roadmap

This roadmap turns the knowledge base into a practical learning and practice plan.

## Week 1: Core Backend

Study:

- .NET execution model
- garbage collection
- C# type system
- async / await
- ASP.NET Core request pipeline
- dependency injection lifetimes
- EF Core performance
- SQL indexes

Practice:

- Build a small orders API.
- Add JWT authentication.
- Add EF Core migrations.
- Add pagination and filtering.
- Explain every design choice out loud.

## Week 2: Frontend And Full-stack Flow

Study:

- TypeScript advanced types
- React hooks
- React rendering model
- REST API design
- frontend loading/error/empty states
- token storage trade-offs

Practice:

- Build an orders list page.
- Build create/edit forms.
- Use React Query.
- Add route protection.
- Add typed API client.

## Week 3: Engineering Depth

Study:

- Redis caching
- Kafka event streaming
- backend performance
- troubleshooting method
- Clean Architecture
- ADRs

Practice:

- Add Redis cache to product/category APIs.
- Add an outbox table.
- Simulate Kafka event publishing.
- Write one ADR for Redis and one ADR for modular monolith.
- Troubleshoot a deliberately slow query.

## Week 4: System Design And Reflection

Study:

- system design method;
- rate limiter;
- notification system;
- file upload system;
- payment callback;
- multi-tenancy;
- collaboration reflection.

Practice:

- Work through 5 system design knowledge checks.
- Write 3 project case notes.
- Write 2 incident case notes.
- Write 1 conflict or collaboration reflection.
- Write 1 architecture trade-off note.

## Daily Speaking Practice

For each topic, answer:

1. What is it?
2. Why does it matter?
3. How is it used in real systems?
4. What can go wrong?
5. How would you debug it?
6. What trade-offs exist?

## Final Learning Narrative

Your engineering narrative should sound like:

> I can build features end to end, and I also think about reliability, maintainability, security, and performance. I care about clear architecture, practical trade-offs, and production operations.
